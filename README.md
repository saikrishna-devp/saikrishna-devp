## Featured Projects

### 🏥 [Healthcare Claims ETL Pipeline](https://github.com/saikrishna-devp/healthcare-claims-etl)
> End-to-end ETL pipeline processing 500K pharmacy claims daily into a PostgreSQL data warehouse

**Tech:** Python · PySpark · Apache Airflow · PostgreSQL · Great Expectations · Docker

- Processes 500K records through multi-stage ETL pipeline with 99.9% uptime
- Star schema dimensional model with fact_claims + 3 dimension tables
- 8 automated data quality checks before every load
- HIPAA compliant - member IDs hashed with SHA256
- Airflow DAG with retry logic and SLA alerts

---

### 📈 [Real-Time Financial Streaming Pipeline](https://github.com/saikrishna-devp/financial-streaming-pipeline)
> Live stock market data streaming via Alpha Vantage API through Kafka into PostgreSQL with live dashboard

**Tech:** Apache Kafka · Python · Alpha Vantage API · PostgreSQL · Plotly Dash · Docker

- Fetches real AAPL, GOOGL, MSFT, AMZN, TSLA prices every 60 seconds
- Kafka producer/consumer architecture with 6 data quality checks
- Live Plotly dashboard updating every 30 seconds
- Dual storage - PostgreSQL primary + SQLite backup
- Flags stocks with price change greater than $5

---

### 💼 [HR Payroll Analytics Data Warehouse](https://github.com/saikrishna-devp/hr-payroll-dwh)
> Cloud data warehouse for 10K employees with real US payroll tax calculations and Tableau dashboard

**Tech:** Python · dbt · PostgreSQL · Tableau Public · Docker

- 10,000 employees across 10 departments with 12 months payroll history
- Real US tax calculations - Federal 22%, State 5%, Social Security 6.2%, Medicare 1.45%
- 102,592 monthly payroll records loaded to PostgreSQL
- dbt 3-layer architecture - raw, staging, marts
- 8 employee + 6 payroll data quality checks
- Live Tableau Dashboard: [View Here](https://public.tableau.com/views/HR_Dashboard_Analytics_17757057726400/Dashboard1)
